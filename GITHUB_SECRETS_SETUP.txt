=================================================================
GitHub Secrets 配置指南
=================================================================

请按照以下步骤操作：

1. 在打开的 GitHub Actions 页面中：
   - 点击 "I understand my workflows, go ahead and enable them" 按钮
   - 确认所有 workflows 显示为绿色 ✓

2. 在 GitHub Secrets 设置页面中，逐个添加以下 secrets：

-----------------------------------------------------------------
Secret Name (变量名)          | Value (值)
-----------------------------------------------------------------
DEEPSEEK_API_KEY             | sk-xxxxxxxxxxxxxxxx
WECOM_WEBHOOK_URL            | https://qyapi.weixin.qq.com/...
NOTION_API_KEY               | <your-notion-api-key>
NOTION_DATABASE_ID           | <your-notion-database-id>
GITHUB_TOKEN                 | <your-github-personal-access-token>
GITHUB_REPO                  | Beltran12138/industry-feeds
-----------------------------------------------------------------

添加方法：
a. 点击 "New repository secret" 按钮
b. 输入 Name（变量名）
c. 输入 Value（值）
d. 点击 "Add secret" 保存
e. 重复 a-d 添加下一个
