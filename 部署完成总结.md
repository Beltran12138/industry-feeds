#### 1️⃣ 启用 GitHub Actions

由于 Vercel Hobby 计划不支持 Cron Jobs，所有定时任务已迁移到 GitHub Actions。

**操作步骤：**
1. 访问：https://github.com/Beltran12138/industry-feeds/actions

## 🆘 故障排查

### GitHub Actions 不运行？
- 确认已启用 Actions
- 检查 secrets 配置是否正确
- 查看 Actions 日志

### Notion 同步失败？
- 确认 Database ID 正确（32 位十六进制）
- 检查 Notion 集成是否已分享到数据库
- 查看 Vercel Logs

### 推送未收到？
- 检查 Webhook URL 是否正确
- 确认机器人已添加到群聊
- 查看 Vercel Logs
