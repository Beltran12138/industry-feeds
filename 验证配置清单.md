## 🔍 故障排查

### 如果 GitHub Actions 没有运行

**检查项：**
1. 确认已启用 Actions（不是仅查看）
2. 检查 secrets 名称是否完全匹配（区分大小写）
3. 查看 Actions 日志了解详细错误

**解决方法：**
```bash
# 手动触发一次抓取（测试用）
curl -X POST https://alpha-radar-eight.vercel.app/api/scrape?tier=high
```

### 如果 Vercel API 报错

**检查项：**
1. 查看 Vercel Logs
2. 确认环境变量已正确添加
3. 重新部署：`vercel --prod`

**查看日志命令：**
```bash
cd C:\Users\lenovo\alpha-radar
vercel logs --follow
```

### 如果集成不工作

**Notion 同步失败：**
- 确认 Database ID 正确（32 位十六进制）
- 检查 Notion 集成是否已分享到数据库
- 验证 NOTION_API_KEY 是否正确

**GitHub 集成失败：**
- 确认 GITHUB_TOKEN 有足够权限（repo, workflow）
- 检查 GITHUB_REPO 格式（username/repo）
- 验证网络连接
